FAQ

Is ChatFull private?

ChatFull is built with privacy in mind. No accounts, no ads, and no tracking.

How does ChatFull work?

You can either create a chatroom and invite others using a unique code, or join an existing room using a code shared with you.

How do I create a chatroom?

Click Create Chatroom, choose a name, and get a unique room code to share with others.

How do I join a chatroom?

Click Join Chatroom, enter the room code, and start chatting instantly.
